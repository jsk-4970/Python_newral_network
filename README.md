# Python 中級 - ニューラルネットワークと強化学習

機械学習とゲームAIを学ぶための中級Pythonプロジェクトです。オブジェクト指向プログラミングの基礎から、ニューラルネットワーク、強化学習まで段階的に学習できます。

## プロジェクト構成

```
python_middle/
├── 1/  オブジェクトとクラス
├── 2/  Python応用プログラミング
├── 3/  強化学習 (Q-Learning)
├── 4/  状態の数え上げ
├── 5/  ニューラルネットワーク I（基礎実装）
├── 6/  ニューラルネットワーク II（発展）
└── 7/  Deep Q-Network
```

## 各モジュールの内容

### モジュール 1: オブジェクトとクラス
- クラスとプロパティの基礎
- 三目並べゲームの状態管理

### モジュール 2: Python応用
- 高度なPythonプログラミング技法

### モジュール 3: 強化学習（Q-Learning）
- Q学習アルゴリズムの実装
- ε-greedy探索戦略
- エージェント対エージェントの対戦システム

### モジュール 4: 状態の数え上げ
- ゲーム状態の列挙
- 総当たり探索

### モジュール 5: ニューラルネットワーク I
- 行列演算の実装（`matrix.py`）
- 活性化関数（Sigmoid, ReLU, Linear）
- 順伝播ネットワークの構築

### モジュール 6: ニューラルネットワーク II
- 微分と逆伝播
- 損失関数

### モジュール 7: Deep Q-Network
- ニューラルネットワークとQ学習の統合

## 特徴

- **外部ライブラリ不使用**: NumPy等を使わず、すべてのアルゴリズムをスクラッチで実装
- **段階的学習**: 基礎から応用へ順を追って学習可能
- **実践的な例題**: 三目並べを題材にした強化学習の実装

## 使用技術

- Python 3
- 標準ライブラリのみ（`random`, `enum`, `math`）

## 実行方法

各モジュールのディレクトリに移動して、Pythonファイルを実行します。

```bash
cd python_middle/5
python main.py
```
